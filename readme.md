# React + TS + Vite - Template

## Packages

1. React with TypeScript
2. Vite
3. Axios
4. React Query
5. Zustand
6. Rome (Lint and Formatter)
7. Material UI
8. Pnpm

## Structure

```
- src
    - config/
    - layout/
    - pages/
    - shared/
    - store/
    - App.tsx
    - index.css
    - main.tsx
    - vite-env.d.ts
- .env.example
- index.html
- package.json
- pnpm-lock.yaml
- rome.json
- tsconfig.json
- tsconfig.node.json
- vite.config.ts
```

## Install packages

```
pnpm install
```

## Run the project

```
pnpm run dev
```
